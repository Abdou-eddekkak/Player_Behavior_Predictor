# Player_Behavior_Predictor
Player Behavior AI Dashboard is an end-to-end machine learning web application that analyzes player behavior patterns over time using an LSTM neural network built with PyTorch. The system processes gameplay sequences (actions, items collected, mistakes, progression, and playtime) to classify players into behavior profiles such as Casual, Aggressive, or Explorer. The project includes a FastAPI backend for real-time inference, an interactive frontend dashboard for sequence input and dynamic prediction visualization, and probability-based charts for model interpretability. The application demonstrates sequence modeling, API deployment, frontend integration, and real-time ML prediction workflows.
